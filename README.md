# Gemini Chatbot using LangChain & Streamlit

Live Chat Bot Link : https://chatbot-iyfg9za9puexf98p2unpir.streamlit.app/





This project is a simple AI chatbot built using Google Gemini, LangChain, and Streamlit.

The application takes user input through a web interface and generates responses using the Gemini 1.5 Flash model. The app is deployed on Streamlit Cloud and uses a secure API key stored in Streamlit Secrets.

Technologies used:

* Python
* Streamlit
* LangChain
* Google Gemini

This project demonstrates how to integrate a large language model with a web interface and deploy it online.
